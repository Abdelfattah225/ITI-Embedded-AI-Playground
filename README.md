### Getting started
1. Clone the repo:
```bash
git clone https://github.com/Abdelfattah225/ITI-Python-Playground.git
cd ITI-Python-Playground
```

2. Create and activate a virtual environment:
```bash
python -m venv .venv
# macOS / Linux
source .venv/bin/activate
# Windows
.venv\Scripts\activate
```

3. Install common packages:
```bash
pip install jupyterlab notebook numpy pandas matplotlib seaborn scikit-learn
```

4. Run Jupyter:
```bash
jupyter lab
```

### Run a notebook from the terminal (optional)
```bash
jupyter nbconvert --to notebook --execute --inplace path/to/notebook.ipynb
```
